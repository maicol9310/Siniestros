# ADR-001: Uso de Clean Architecture

## Estado
Aceptado

## Contexto
Se requiere una arquitectura mantenible, testeable y desacoplada.

## Decisión
Implementar Clean Architecture separando:
- Dominio
- Aplicación
- Infraestructura
- API

## Consecuencias
+ Alta mantenibilidad
+ Facilidad para pruebas unitarias
- Mayor número de proyectos
